# Анализ стабильности и невязок паспортных параметров поляризаторов

Этот план описывает шаги для проведения комплексного статистического анализа стабильности фоновых (bg) измерений и невязок (residuals) экспериментальных данных относительно паспортных параметров модели Бланко для всех доступных серий измерений.

## User Review Required

> [!NOTE]
> Анализ будет проводиться для всех 6 серий данных (`356att`, `series1`, `series2`, `series3`, `series4`, `series5`). 
> Паспортные параметры фиксируются на значениях:
> - $P_{\text{eff}} = 15.50$ мкм
> - $D_{\text{eff}} = 5.67$ мкм
> - $\theta_{\text{offset}} = -0.45^\circ$
> - $\alpha = 0.255$ дБ/ТГц$^{1.58}$ ( loss_factor ), $\gamma = 1.58$
> - Шумовой порог eps_floor = 0.0
>
> Оптимизация параметров проводиться не будет, расчет будет быстрым. Результаты анализа стабильности и невязок будут оформлены в виде подробного отчета с графиками.

## Proposed Changes

### [1. Скрипт анализа] [NEW] [analyze_stability_and_residuals.py](file:///c:/THz-Unified-Optimizer/scripts/analyze_stability_and_residuals.py)
Создадим новый скрипт в папке `scripts/` для выполнения расчетов и генерации графиков:
- **Анализ стабильности bg**:
  - Хронологическая сортировка измерений по времени модификации файлов (или по углам в названии файлов, если mtime совпадает).
  - Вычисление временной интегральной метрики $\int E_{bg}^2(t) dt$.
  - Вычисление спектральной интегральной метрики $\int |S_{bg}(\nu)|^2 d\nu$ и проверка теоремы Парсеваля.
  - Расчет спектрального шумового пола (noise floor) на частотах выше 2.5 ТГц.
  - Расчет интегральной спектральной характеристики в рабочем диапазоне 0.2–1.8 ТГц.
  - Статистика: среднее, СКО, относительное стандартное отклонение (RSD, %), линейный дрейф.
- **Анализ невязок (residuals)**:
  - Сравнение 1D интегрального пропускания (эксперимент vs паспортная Blanco-модель), вычисление линейных и децибельных невязок от угла $\theta$.
  - Сравнение 2D спектрально-углового пропускания (эксперимент vs паспортная Blanco-модель), генерация 2D-карт невязок от угла и частоты в осях $(\nu, \theta)$.
  - Визуализация: сохранение графиков стабильности и карт невязок в `docs/images/`.
