# План полномасштабного исследования поляризатора 40/20 мкм

## Контекст и постановка задачи

**Новый эксперимент**: Поляризатор (P=40 мкм, D=20 мкм) заключён **между двумя практически идеальными плёночными поляризаторами**.

Это принципиально отличается от прежней конфигурации, где система состояла из **двух одинаковых WGP-решёток** (P=15.5 мкм). Схема:

```
[Пленочный поляризатор P1] → [ТГц пучок] → [WGP (P=40, D=20)] → [Пленочный поляризатор P2]
```

Матрица Джонса для старой конфигурации (два одинаковых WGP):
```
M_old = P_wgp · R(θ) · P_wgp · R⁻¹(θ)
```

**Матрица Джонса для новой конфигурации** (идеальные поляризаторы до и после WGP):
```
M_new = P_film · R(-φ₂) · P_wgp · R(θ) · P_film · R(-φ₁)
```

Где:
- `P_film` — матрица **идеального** плёночного поляризатора: передаёт только одну линейную компоненту, полностью гасит ортогональную.
- `P_wgp` — матрица WGP с параметрами t_perp(ν) и t_par(ν) из модели Бланко-Друде.
- `θ` — угол поворота WGP (измеряемая переменная, от −100° до +100°).
- `φ₁, φ₂` — фиксированные ориентации внешних поляризаторов (равны 0°, если оба выровнены вдоль оси x).

> [!IMPORTANT]
> При идеальных внешних поляризаторах (`φ₁ = φ₂ = 0`) матрица системы существенно упрощается:
> `E_out_x = t_perp(ν) · cos²(θ) + t_par(ν) · sin²(θ)` (закон Малюса для WGP)
> Это принципиально другая функциональная зависимость от θ, чем в старом эксперименте.

> [!IMPORTANT]
> **Вопрос 1**: Как именно ориентированы внешние плёночные поляризаторы относительно угла θ WGP?
> Есть ли свободный угловой offset у одного из них, или оба жёстко зафиксированы?

> [!IMPORTANT]
> **Вопрос 2**: Что служит «фоновым» (reference) измерением (bg-файлы)? Сигнал с двумя плёночными поляризаторами без WGP, или сигнал с установленным WGP при θ=0°?

> [!NOTE]
> **Вопрос 3**: D/P = 20/40 = 0.5 — это ровно на границе режима «плотной» решётки. Ожидается ли, что D_eff ≈ D_phys = 20 мкм, в отличие от старого поляризатора, где D_eff ≪ D_phys?
