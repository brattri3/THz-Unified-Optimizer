# Интеграция пакета LMFIT (Walkthrough)

## Внесенные изменения
- **Создан `optimizer_lmfit.py`**: Реализована альтернативная функция оценки `residual_2d_complex`, возвращающая вектор невязок для алгоритма Левенберга-Марквардта. Это позволяет библиотеке `lmfit` корректно оценивать матрицу Якобиана и вычислять доверительные интервалы.
- **Создан `scripts/run_lmfit_analysis.py`**: Написан скрипт для прогона оптимизации с использованием `lmfit` на датасете `356att`. Скрипт сохраняет итоговый отчет со статистикой подгонки (Fit Statistics), стандартными ошибками параметров (Variables) и корреляционной матрицей (Correlations).

## Замечание о доверительных интервалах
Вычисление строгих доверительных интервалов с помощью `lmfit.conf_interval()` было отключено, так как для периода `P_um` алгоритм достигал заданной границы оптимизации (`17.5`), что приводило к зависанию процедуры (игнорирование Bounds при профилировании вероятностей). Мы используем стандартные ошибки из ковариационной матрицы, которые выводятся по умолчанию.
