# План создания мастер-документа: Полная спецификация и статистический анализ модели WGP

## Цель работы

Создать **фундаментальный, математически и физически строгий документ** (`complete_model_specification.md`), исчерпывающе описывающий текущую модель THz-TDS для проволочных поляризаторов (WGP). 

Документ должен содержать:
1. Полный набор уравнений (модель Бланко, импеданс Друде, модель рассеяния, анизотропные фазовые задержки, формализм матриц Джонса).
2. Полный перечень всех параметров модели с физическим обоснованием необходимости их введения.
3. Количественные доказательства влияния каждого параметра через **Ablation Study** ($\chi^2_\nu$, RMSE амплитуды/фазы, AIC, BIC, $\Delta\text{AIC}$).
4. Комплексный анализ остатков (невязок): проверка нормальности (Шапиро-Уилк, Харке-Бера), Q-Q графики, 2D-карты невязок (по частоте и углу), физический анализ причин ненормальности (апертурная дифракция при скрещивании).
5. Графические доказательства для каждого этапа усложнения модели.

## Структура мастер-документа `complete_model_specification.md`

### Раздел 1. Физико-математический формализм полной модели
- **1.1. Модель эквивалентных схем Бланко (Pure Blanco)**: уравнения для $t_{\perp}(\nu)$ и $t_{\parallel}(\nu)$, гармоники рядов Бланко $A_1, A_2, f_a, f_b, f_c, f_d$.
- **1.2. Высокочастотный омический импеданс Друде**: формализм поверхностного импеданса $Z_s(\omega) = \sqrt{\frac{i \omega \mu_0}{\sigma(\omega)}}$, параметры вольфрама ($\sigma_0 = 1.8\times 10^7$ См/м, $\tau = 8$ фс).
- **1.3. Модель диффузного рассеяния (Theory of Perturbations)**: феноменологическая экспоненциальная поправка $e^{-\frac{1}{2} \alpha \nu^\gamma}$.
- **1.4. Фазовая анизотропия метаповерхности**: разделение задержек $\tau_{\text{ps}}$ (для TM) и $\tau_{\text{par\_ps}}$ (для TE).
- **1.5. Формализм матриц Джонса**:
  - Конфигурация A (два одинаковых WGP): $M = P_{\text{wgp}} R(\theta) P_{\text{wgp}} R^{-1}(\theta)$.
  - Конфигурация B (WGP между плёнками): $M = P_{\text{film}} R(\phi_2) P_{\text{wgp}}(\theta) P_{\text{film}} R(-\phi_1)$.
- **1.6. Закономерность эффективного диаметра**: формула эмпирической оценки $D_{\text{eff}} = D_{\text{phys}} \left(1 - 0.85 \frac{D_{\text{phys}}}{P}\right)$.

- Физическая причина введения

### Раздел 3. Доказательный Ablation Study (включение/отключение параметров)
Расчёт и сравнение 5 модификаций модели на ключевых датасетах (`356att` и `test_grid_40_20`):
- **M0**: Pure Blanco (только геометрическая эквивалентная схема)
- **M1**: Blanco + Drude
- **M2**: Blanco + Drude + Scattering ($\gamma = 2$)
- **M3**: Blanco + Drude + Scattering + $\tau_{\text{par}}$ (фазовая анизотропия)
- **M4**: Blanco + Drude + Scattering (free $\gamma$) + $\tau_{\text{par}}$

Метрики для каждой модели:
- $D_{\text{eff}}$
- $\chi^2_\nu$ (приведённый хи-квадрат)
- $\text{RMSE}_{\text{amp}}$ (дБ)
- $\text{RMSE}_{\text{phase}}$ (рад)
- AIC, BIC, $\Delta\text{AIC}$

### Раздел 4. Строгий статистический анализ остатков (Residuals)
- **4.1. Тесты нормальности**: результаты критериев Шапиро-Уилка и Харке-Бера для амплитудных и фазовых невязок.
- **4.2. Анализ Q-Q графиков**: квантиль-квантильный анализ тяжёлых хвостов (отклонения в режимах глубокого гашения).
- **4.3. 2D-карты невязок**: распределение ошибок по частоте (0.3–2.5 ТГц) и углу ($\theta$).
- **4.4. Физическая природа остаточной несистемной ошибки**: апертурная дифракция пучка на оправе поляризатора и пределы аналитического приближения бесконечной плоской решётки.
